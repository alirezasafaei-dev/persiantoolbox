# راهنمای کاربری جعبه ابزار فارسی

**نسخه ۶.۷.۰** | ۷۶ ابزار | ۴۳۴ تست

## معرفی

جعبه ابزار فارسی (persiantoolbox.ir) مجموعه‌ای کامل از ابزارهای آنلاین برای کاربران فارسی‌زبان است. تمام پردازش‌ها در مرورگر انجام می‌شود و هیچ داده‌ای به سرور ارسال نمی‌شود.

---

## جستجوی ابزارها

از نوار جستجو در بالای صفحه اصلی برای پیدا کردن سریع ابزار مورد نظر خود استفاده کنید. جستجو بر اساس نام فارسی ابزار، توضیحات و کلمات کلیدی انجام می‌شود.

## ابزارهای متنی (۱۱ ابزار)

### شمارنده کلمات و کاراکترها (`/text-tools/word-counter`)

- شمارش آنی کلمات، کاراکترها، جملات و پاراگراف‌ها
- پشتیبانی از فارسی و انگلیسی
- نمایش آنی آمار

### تبدیل اعداد (`/text-tools/number-converter`)

- تبدیل آنی اعداد فارسی (۰۱۲۳۴۵۶۷۸۹) به انگلیسی (0123456789) و بالعکس
- پشتیبانی از اعداد عربی
- کپی سریع نتیجه

### حذف فاصله‌ها (`/text-tools/remove-spaces`)

- حذف فاصله‌های اضافی و Tab
- حذف فاصله انتهای خطوط
- حذف همه فاصله‌ها

### تبدیل Case (`/text-tools/case-converter`)

- تبدیل به حروف بزرگ (UPPER)
- تبدیل به حروف کوچک (lower)
- تبدیل به حالت عنوانی (Title)
- تبدیل به حالت جمله‌ای (Sentence)
- تبدیل معکوس (tOGGLE)

### استخراج اطلاعات (`/text-tools/extract-info`)

- استخراج خودکار ایمیل از متن
- استخراج شماره تلفن
- استخراج لینک‌ها و URL
- استخراج اعداد

### ساخت رزومه آنلاین (`/text-tools/resume-builder`)

- ساخت رزومه حرفه‌ای فارسی
- پیش‌نمایش زنده و خروجی PDF
- RTL کامل

### امضای آنلاین (`/text-tools/signature`)

- امضای دیجیتال با ماوس یا لمس
- خروجی PNG با پس‌زمینه شفاف
- ذخیره‌سازی محلی

### تبدیل آدرس فارسی به انگلیسی (`/text-tools/address-fa-to-en`)

- تبدیل استاندارد آدرس فارسی به فرمت انگلیسی
- مناسب برای ارسال پستی و فرم‌های بین‌المللی
- خروجی تک‌خطی و چندخطی

### فرمت‌بندی JSON (`/tools/json-formatter`)

- فرمت‌بندی، اعتبارسنجی و فشرده‌سازی JSON
- کاملاً محلی بدون ارسال داده

### تولید هش (`/tools/hash-generator`)

- تولید هش با الگوریتم‌های SHA-1/256/384/512
- کاملاً محلی و امن

### رمزگذاری Base64 (`/tools/base64-tool`)

- تبدیل متن به Base64 و بالعکس
- کاملاً محلی

## نکات فنی

- **پردازش محلی**: تمام محاسبات و پردازش‌ها در مرورگر شما انجام می‌شود
- **حریم خصوصی**: هیچ داده‌ای به سرور ارسال نمی‌شود
- **بدون ثبت‌نام**: بیشتر ابزارها بدون نیاز به حساب کاربری قابل استفاده هستند
- **فرمت‌های پشتیبانی**: PDF, JPG, PNG, WebP, DOC, DOCX
- **حداکثر حجم فایل**: ۵۰ مگابایت
- **سازگاری**: Chrome, Firefox, Safari, Edge (نسخه‌های مدرن)
- **RTL**: رابط کاربری کاملاً راست‌چین با پشتیبانی حالت تاریک
