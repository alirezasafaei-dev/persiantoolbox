# راهنمای کاربری جعبه ابزار فارسی

## معرفی

جعبه ابزار فارسی مجموعه‌ای کامل از ابزارهای آنلاین برای کاربران فارسی‌زبان است. تمام پردازش‌ها در مرورگر انجام می‌شود و هیچ داده‌ای به سرور ارسال نمی‌شود.

## ابزارهای متنی

### شمارنده کلمات (`/text-tools/word-counter`)

- شمارش کلمات، کاراکترها، جملات و پاراگراف‌ها
- پشتیبانی از فارسی و انگلیسی
- نمایش آنی آمار

### تبدیل اعداد (`/text-tools/number-converter`)

- تبدیل اعداد فارسی (۰۱۲۳۴۵۶۷۸۹) به انگلیسی (0123456789)
- تبدیل اعداد عربی (٠١٢٣٤٥٦٧٨٩)
- کپی سریع نتیجه

### حذف فاصله‌ها (`/text-tools/remove-spaces`)

- حذف فاصله‌های اضافی
- حذف فاصله انتهای خطوط
- حذف همه فاصله‌ها

### تبدیل Case (`/text-tools/case-converter`)

- تبدیل به حروف بزرگ (UPPER)
- تبدیل به حروف کوچک (lower)
- تبدیل به حالت عنوانی (Title)
- تبدیل به حالت جمله‌ای (Sentence)
- تبدیل معکوس (tOGGLE)

### استخراج اطلاعات (`/text-tools/extract-info`)

- استخراج ایمیل از متن
- استخراج شماره تلفن
- استخراج لینک‌ها
- استخراج اعداد

## ابزارهای تاریخ

### تبدیل تاریخ (`/date-tools/shamsi-gregorian`)

- تبدیل تاریخ شمسی به میلادی
- تبدیل تاریخ میلادی به شمسی
- پشتیبانی از تمام سال‌ها

### محاسبه差异 تاریخ (`/date-tools/date-difference`)

- محاسبه تعداد روز بین دو تاریخ
- نمایش به صورت هفته، ماه و سال

### تقویم فارسی (`/date-tools/persian-calendar`)

- نمایش ماه‌های فارسی
- نمایش روز جاری
- پشتیبانی از سال‌های کبیسه

### یادآوری رویدادها (`/date-tools/event-reminder`)

- ثبت رویدادهای مهم
- نمایش رویدادهای آینده و گذشته
- ذخیره در مرورگر

## ابزارهای تصویر

### چرخش تصویر (`/image-tools/rotate-image`)

- چرخش با زوایای مختلف (90°، 180°، 270°)
- چرخش با زاویه دلخواه
- دانلود تصویر خروجی

### تغییر اندازه (`/image-tools/resize-image`)

- تغییر عرض و ارتفاع
- حفظ نسبت تصویر
- نمایش حجم اصلی و خروجی

### افزودن متن (`/image-tools/text-on-image`)

- انتخاب رنگ و اندازه فونت
- انتخاب موقعیت متن (بالا، وسط، پایین)
- دانلود تصویر خروجی

- محاسبه مالیات بر درآمد ۱۴۰۵

- جدول پلکانی مالیاتی

### مقایسه نرخ سود (`/tools/bank-rate-comparator`)

- مقایسه نرخ سود بانک‌ها
- محاسبه سود ماهانه و سالانه
- مرتب‌سازی بر اساس نرخ سود

### محاسبه هزینه زندگی (`/tools/living-cost`)

- تخمین هزینه‌های ماهانه
- دسته‌بندی: مسکن، خوراک، حمل‌ونقل، قبوض
- محاسبه هزینه سالانه
