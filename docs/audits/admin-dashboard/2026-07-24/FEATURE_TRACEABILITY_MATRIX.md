## Status Legend

- **VERIFIED_FUNCTIONAL**: Full chain from UI to persistence verified
- **PARTIALLY_CONNECTED**: Some data real, some missing or stale
- **UI_ONLY**: UI exists but no real backend connection
- **EXTERNAL_BLOCKER**: Depends on external service credential/config
- **REMOVED**: Feature removed during audit
