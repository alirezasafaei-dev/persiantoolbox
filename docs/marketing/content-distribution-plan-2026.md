# PersianToolbox Content Distribution Program — 2026

> برنامه ۱۲ هفته‌ای انتشار محتوای آموزشی و غیرتبلیغاتی برای معرفی PersianToolbox و ابزارهای آن.

## اهداف

- انتشار ۱۲ مقاله مرجع ابتدا روی PersianToolbox
- بازآفرینی متناسب برای ۱۰ پلتفرم خارجی و ۱۰ پلتفرم ایرانی
- توزیع خلاصه و محتوای قابل ذخیره در پیام‌رسان‌ها
- سنجش Referral، Engagement، Tool Start و Tool Completion
- جلوگیری از انتشار تکراری، خودمعرفی افراطی و لینک‌سازی اسپمی

## اصول غیرقابل‌مذاکره

1. نسخه اصلی مقاله ابتدا روی PersianToolbox منتشر شود.
2. Medium، DEV و Hashnode فقط با canonical یا Original URL استفاده شوند.
3. LinkedIn، Quora، Reddit و پلتفرم‌های ایرانی نسخه بازنویسی‌شده دریافت کنند.
4. هر مقاله حتی بدون لینک باید مسئله کاربر را حل کند.
5. هر نسخه حداکثر یک لینک طبیعی به ابزار مرتبط داشته باشد.
6. از ادعاهای «بهترین»، «بی‌رقیب» و CTAهای تبلیغاتی خودداری شود.
7. نتیجه هر کانال در روزهای ۷ و ۲۸ ارزیابی شود.

## قاعده ادامه یا توقف

- **افزایش سهم:** Completion Rate حداقل ۱۵٪ یا تعامل کیفی بالا
- **بهینه‌سازی:** ورودی وجود دارد ولی شروع یا تکمیل ابزار پایین است
- **بازنگری محتوا:** ورودی و تعامل هر دو ضعیف‌اند
- **توقف آزمایشی:** پس از سه انتشار همچنان ورودی یا تعامل معنادار وجود ندارد
