# راهنمای تصاویر بلاگ

استانداردهای استفاده از تصاویر در مقالات بلاگ جعبه ابزار فارسی.

## مسیر ذخیره‌سازی

```
public/images/blog/<article-slug>/
```

مثال:

```
public/images/blog/loan-calculation/cover.webp
public/images/blog/loan-calculation/step-1.webp
public/images/blog/loan-calculation/result.webp
```

## نام‌گذاری فایل‌ها

| نام              | کاربرد                        |
| ---------------- | ----------------------------- |
| `cover.webp`     | تصویر شاخص مقاله (hero image) |
| `step-1.webp`    | مرحله اول آموزش               |
| `step-2.webp`    | مرحله دوم آموزش               |
| `result.webp`    | نتیجه نهایی                   |
| `checklist.webp` | چک‌لیست یا خلاصه              |

## ابعاد پیشنهادی

| نوع تصویر               | عرض           | ارتفاع | نسبت    |
| ----------------------- | ------------- | ------ | ------- |
| cover (شاخص)            | 1200px        | 630px  | ~1.91:1 |
| screenshot (اسکرین‌شات) | حداکثر 1400px | —      | —       |
| inline diagram (نمودار) | 1200px        | 800px  | 3:2     |

## فرمت و حجم

| نوع        | فرمت         | حجم هدف       |
| ---------- | ------------ | ------------- |
| cover      | WebP یا AVIF | کمتر از 180KB |
| screenshot | WebP یا AVIF | کمتر از 250KB |
| diagram    | WebP یا AVIF | کمتر از 150KB |

ابزارهای پیشنهادی برای فشرده‌سازی:

- [Squoosh](https://squoosh.app/) — آنلاین، رایگان
- [Sharp](https://sharp.pixelplumbing.com/) — برنامه‌نویسی

## قوانین Alt Text فارسی

1. **توصیفی باشد** — محتوای تصویر را به فارسی شرح دهد
2. **کلیدواژه اصلی** — حداقل یک کلیدواژه مرتبط داشته باشد
3. **بلند نباشد** — حداکثر ۱۲۵ کاراکتر
4. **تکراری نباشد** — alt هر تصویر با بقیه متفاوت باشد

مثال خوب:

```yaml
coverAlt: 'نمونه محاسبه اقساط وام در جعبه ابزار فارسی'
```

مثال بد:

```yaml
coverAlt: 'تصویر'
```

## قوانین حریم خصوصی

- **هرگز** از داده واقعی کاربران استفاده نکنید
- **هرگز** اسکرین‌شات حاوی اطلاعات شخصی منتشر نکنید
- فقط از داده‌های نمونه (dummy data) استفاده کنید
- نام‌ها، شماره‌ها و آدرس‌ها باید ساختگی باشند

## استفاده از تصویر در Markdown

### تصویر شاخص (Hero Image)

تصویر شاخص از طریق `coverImage` در frontmatter تعریف می‌شود و نیازی به نوشتن آن در بدنه مقاله نیست.

### تصاویر داخل مقاله

```markdown
![توضیح تصویر](/images/blog/loan-calculation/step-1.webp)

_شکل ۱: مراحل ورود اطلاعات در ابزار محاسبه وام_
```

نکات:

- **Alt text** را به فارسی بنویسید
- **کپشن** با italic بعد از تصویر
- تصاویر با `![alt](path)` ساده نوشته شوند
- حداکثر عرض تصاویر داخل مقاله: 100% (توسط CSS مدیریت می‌شود)

### مثال کامل Markdown

```markdown
## مرحله اول: ورود اطلاعات

اطلاعات وام خود را در فیلدهای زیر وارد کنید:

![ورود اطلاعات وام](/images/blog/loan-calculation/step-1.webp)

_شکل ۱: فرم ورود اطلاعات وام در ابزار محاسبه اقساط_

## مرحله دوم: مشاهده نتیجه

پس از ورود اطلاعات، جدول بازپرداخت کامل نمایش داده می‌شود:

![جدول بازپرداخت وام](/images/blog/loan-calculation/result.webp)

_شکل ۲: جدول بازپرداخت اقساط با جزئیات کامل_
```

## چک‌لیست قبل از انتشار

- [ ] تصویر cover با ابعاد 1200x630 و فرمت webp
- [ ] حجم cover کمتر از 180KB
- [ ] alt text فارسی و توصیفی برای همه تصاویر
- [ ] کپشن مناسب برای تصاویر داخل مقاله
- [ ] هیچ داده واقعی کاربران در تصاویر نیست
- [ ] مسیر تصاویر در frontmatter صحیح است
- [ ] تصاویر در مرورگر تست شده‌اند
