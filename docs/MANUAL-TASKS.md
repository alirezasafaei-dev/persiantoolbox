## راهنمای ورود به پنل ادمین

### مراحل ورود:

1. **به سایت بروید:** https://persiantoolbox.ir/account
2. **ثبت‌نام کنید:** ایمیل و رمز عبور دلخواه وارد کنید
3. **ایمیل خود را به من بدهید** تا در متغیرهای محیطی VPS اضافه کنم
4. **وارد شوید** با همان ایمیل و رمز
5. **به آدرس بروید:** https://persiantoolbox.ir/admin

### نحوه فعال‌سازی دسترسی ادمین:

ایمیل شما باید در متغیر `ADMIN_EMAIL_ALLOWLIST` در فایل `.env` روی VPS اضافه شود:

```bash
ssh ubuntu@193.93.169.32
nano /home/ubuntu/persiantoolbox/.env
# اضافه کردن: ADMIN_EMAIL_ALLOWLIST=your-email@example.com
pm2 restart persiantoolbox
```

### صفحات ادمین موجود:

- `/admin` — داشبورد اصلی
- `/admin/analytics` — آمار و تحلیل
- `/admin/site-settings` — تنظیمات سایت (ایمیل، تلفن، شبکه اجتماعی)
- `/admin/users` — مدیریت کاربران
- `/admin/ops` — مانیتورینگ عملیاتی
- `/admin/content` — مدیریت محتوا
- `/admin/monetization` — مدیریت درآمدزایی
- `/admin/tools` — مدیریت ابزارها

## کارهای دستی مورد نیاز

### ۱. دپلوی بات تلگرام

**وضعیت:** کد آماده در `packages/telegram-bot/` | بات با موفقیت تست شد (`@persiantoolbox_bot`)

**موضوع:** سرور VPS (ایران) به API تلگرام دسترسی ندارد (DNS فیلتر). بات در حال اجرا روی **دستگاه توسـعه** با PM2 است.

**مدیریت بات محلی:**

```bash
# استاتوس
pm2 list
# لاگ‌ها
pm2 logs persiantoolbox-telegram-bot
# ری‌استارت
pm2 restart persiantoolbox-telegram-bot
# استاپ
pm2 stop persiantoolbox-telegram-bot
```

**اقدامات ممکن:**

- راهکار دائم: خرید سرور خارج از ایران (آلمان/هلند) و انتقال بات به آنجا
- راهکار جایگزین: استفاده از پروکسی SOCKS5 در کد بات (نیاز修改 `bot.js`)
- **فعلاً:** بات روی دستگاه توسـعه اجرا می‌شود و نیازی به اقدام ندارد

### ۲. انتشار Chrome Extension

**وضعیت:** کد کامل در `packages/chrome-extension/` | آماده انتشار در Chrome Web Store

**مراحل انتشار:**

1. ساخت ZIP از `packages/chrome-extension/`
2. رفتن به https://chrome.google.com/webstore/devconsole
3. ثبت‌نام توسعه‌کننده (یکبار هزینه $5)
4. آپلود فایل ZIP + تکمیل اطلاعات
5. ارسال برای بررسی (معمولاً 1-2 روز)

**نکات مهم:**

- آیکون‌ها در `icons/` باید سایزهای 16x16, 48x48, 128x128 داشته باشند
- `manifest.json` آماده است - فقط عکس‌ها را جایگزین کنید
- اکستنشن فقط از `https://persiantoolbox.ir/*` می‌تواند دسترسی داشته باشد
- API key در فایل `.env` برای extension نیست (اکستنشن مستقل است)
- نام پیشنهادی: "جعبه ابزار فارسی — ابزارهای سریع در یک کلیک"

### ۳. درگاه پرداخت زرین‌پال

- **وضعیت:** کد آماده است، دامنه در زرین‌پال ثبت شده، منتظر تایید
- **اقدام:** بعد از تایید زرین‌پال، در داشبورد زرین‌پال بررسی کنید که آیا تراکنش آزمایشی انجام می‌شود
- **تست:** یک پرداخت آزمایشی از صفحه `/pricing` انجام دهید

### ۴. ثبت‌نام در Google Search Console

- **آدرس:** https://search.google.com/search-console
- **دامنه:** persiantoolbox.ir
- **اقدام:** مالکیت دامنه را تایید کنید (از روش DNS TXT)
- **سپس:** نقشه سایت را اضافه کنید: `https://persiantoolbox.ir/sitemap.xml`
- **URL Inspection:** چند صفحه کلیدی را بررسی و درخواست ایندکس کنید

### ۵. ثبت‌نام در Bing Webmaster Tools

- **آدرس:** https://www.bing.com/webmasters
- **وضعیت:** سایت‌مپ قبلاً ایندکس شده
- **اقدام:** URL Submission برای صفحات کلیدی:
  - `/` (صفحه اصلی)
  - `/tools` (ابزارهای مالی)
  - `/loan` (محاسبه وام)
  - `/salary` (محاسبه حقوق)
  - `/pdf-tools` (ابزارهای PDF)
  - `/date-tools/shamsi-gregorian` (تبدیل تاریخ)
  - ۳ پست pillar جدید

### ۶. تنظیمات پنل ادمین

- **ایمیل پشتیبانی:** از پنل ادمین (`/admin/site-settings`) بررسی کنید
- **شماره تلفن:** بررسی کنید
- **آدرس:** بررسی کنید
- **لینک تلگرام:** اگر کانال تلگرام دارید اضافه کنید
- **لینک اینستاگرام:** اگر اکانت دارید اضافه کنید
