**تاریخ:** 2026-06-28

## راهنمای ورود به پنل ادمین

### مراحل ورود:

1. **به سایت بروید:** https://persiantoolbox.ir/account
2. **ثبت‌نام کنید:** ایمیل و رمز عبور دلخواه وارد کنید
3. **ایمیل خود را به من بدهید** تا در متغیرهای محیطی VPS اضافه کنم
4. **وارد شوید** با همان ایمیل و رمز
5. **به آدرس بروید:** https://persiantoolbox.ir/admin

### نحوه فعال‌سازی دسترسی ادمین:

ایمیل شما باید در متغیر `ADMIN_EMAIL_ALLOWLIST` در فایل `.env` روی VPS اضافه شود:

```bash
ssh ubuntu@193.93.169.32
nano /home/ubuntu/persiantoolbox/.env
# اضافه کردن: ADMIN_EMAIL_ALLOWLIST=your-email@example.com
pm2 restart persiantoolbox
```

### صفحات ادمین موجود:

- `/admin` — داشبورد اصلی
- `/admin/analytics` — آمار و تحلیل
- `/admin/site-settings` — تنظیمات سایت (ایمیل، تلفن، شبکه اجتماعی)
- `/admin/users` — مدیریت کاربران
- `/admin/ops` — مانیتورینگ عملیاتی
- `/admin/content` — مدیریت محتوا
- `/admin/monetization` — مدیریت درآمدزایی
- `/admin/tools` — مدیریت ابزارها

## کارهای دستی مورد نیاز

### ۱. درگاه پرداخت زرین‌پال

- **وضعیت:** کد آماده است، دامنه در زرین‌پال ثبت شده، منتظر تایید
- **اقدام:** بعد از تایید زرین‌پال، در داشبورد زرین‌پال بررسی کنید که آیا تراکنش آزمایشی انجام می‌شود
- **تست:** یک پرداخت آزمایشی از صفحه `/pricing` انجام دهید

### ۲. ثبت‌نام در Google Search Console

- **آدرس:** https://search.google.com/search-console
- **دامنه:** persiantoolbox.ir
- **اقدام:** مالکیت دامنه را تایید کنید (از روش DNS TXT)
- **سپس:** نقشه سایت را اضافه کنید: `https://persiantoolbox.ir/sitemap.xml`
- **URL Inspection:** چند صفحه کلیدی را بررسی و درخواست ایندکس کنید

### ۳. ثبت‌نام در Bing Webmaster Tools

- **آدرس:** https://www.bing.com/webmasters
- **وضعیت:** سایت‌مپ قبلاً ایندکس شده (طبق گفته شما)
- **اقدام:** URL Submission برای صفحات کلیدی:
  - `/` (صفحه اصلی)
  - `/tools` (ابزارهای مالی)
  - `/loan` (محاسبه وام)
  - `/salary` (محاسبه حقوق)
  - `/pdf-tools` (ابزارهای PDF)
  - `/date-tools/shamsi-gregorian` (تبدیل تاریخ)
  - ۳ پست pillar جدید

### ۴. تنظیمات پنل ادمین

- **ایمیل پشتیبانی:** از پنل ادمین (`/admin/site-settings`) بررسی کنید درست است
- **شماره تلفن:** بررسی کنید
- **آدرس:** بررسی کنید
- **لینک تلگرام:** اگر کانال تلگرام دارید، لینک را اضافه کنید
- **لینک اینستاگرام:** اگر اکانت اینستاگرام دارید، لینک را اضافه کنید

### ۵. Google Analytics (اختیاری)

- **وضعیت:** سیستم تحلیل خودمیزبان فعال است
- **اقدام:** اگر می‌خواهید GA4 اضافه کنید، کد اندازه‌گیری را در پنل ادمین (`/admin/site-settings` → بخش تحلیل) وارد کنید

### ۶. بررسی عملکرد سایت

- **Lighthouse:** ابزار Lighthouse Chrome را روی صفحه اصلی اجرا کنید
- **Core Web Vitals:** در Google Search Console بررسی کنید
- **Mobile Friendly:** در Google Search Console بررسی کنید

### ۷. محتوای بلاگ

- **وضعیت:** ۵۷ مقاله موجود است
- **اقدام:** مقالات موجود را بررسی و ویرایش کنید
- **مقاله جدید:** هر هفته ۱-۲ مقاله باکیفیت اضافه کنید
- **موضوعات پیشنهادی:**
  - راهنمای جامع ابزارهای تصویر فارسی
  - راهنمای جامع ابزارهای متنی فارسی
  - راهنمای جامع ابزارهای اعتبارسنجی فارسی
  - مقایسه ابزارهای فارسی با ابزارهای انگلیسی

### ۸. شبکه‌های اجتماعی

- **تلگرام:** کانال تلگرام فعال است (@persiantoolbox)
- **اینستاگرام:** اگر اکانت دارید، لینک را در پنل ادمین اضافه کنید
- **توییتر:** اگر اکانت دارید، لینک را در پنل ادمین اضافه کنید

### ۹. بررسی امنیتی

- **SSL:** گواهی Let's Encrypt فعال است — بررسی کنید منقضی نشده
- **SSH:** کلید SSH فعال است
- **UFW:** فایروال فعال است
- **fail2ban:** فعال است

### ۱۰. پشتیبان‌گیری

- **وضعیت:** پشتیبان‌گیری خودکار روزانه فعال است
- **اقدام:** یک پشتیبان دستی انجام دهید و مطمئن شوید کار می‌کند:
  ```bash
  cd /home/dev13/my-project/sites/live/persiantoolbox
  uv run python -m automation.backup
  ```
