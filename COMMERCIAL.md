# Commercial Terms

## Overview

PersianToolbox offers dual-licensing model:

- **Non-commercial use**: MIT License (free)
- **Commercial use**: Requires commercial license for v2.0.0+

## When is a Commercial License Required?

A commercial license is required when:

- Using the software in a commercial product or service
- Charging end users for access to features
- Using the software in enterprise/corporate environments
- White-labeling or removing attribution
- Exceeding 10 active users in a deployment
