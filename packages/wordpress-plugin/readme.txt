== Description ==

ابزارک جعبه ابزار فارسی را به وب‌سایت وردپرسی خود اضافه کنید.
کاربران شما می‌توانند به سرعت به ۸۰+ ابزار رایگان فارسی دسترسی داشته باشند.

ویژگی‌ها:
* جستجوی سریع ابزارها
* دسته‌بندی ۱۰ گروه
* پشتیبانی از پوسته تیره/روشن
* کاملاً رایگان

== Installation ==

1. پوشه `persiantoolbox-widget` را به `/wp-content/plugins/` آپلود کنید
2. پلاگین را از طریق منوی "پلاگین‌ها" در وردپرس فعال کنید
3. به تنظیمات → جعبه ابزار فارسی بروید و تنظیمات را انجام دهید
