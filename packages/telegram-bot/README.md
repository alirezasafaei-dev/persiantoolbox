# جعبه ابزار فارسی - Telegram Bot

بات تلگرام برای دسترسی سریع به ابزارهای آنلاین فارسی.

## نصب

1. با [@BotFather](https://t.me/BotFather) یک بات جدید بسازید
2. توکن بات را دریافت کنید
3. متغیر محیطی `TELEGRAM_BOT_TOKEN` را تنظیم کنید
4. اجرا کنید:

```bash
cd packages/telegram-bot
npm install
npm start
```

## دستورات

| دستور       | توضیح              |
| ----------- | ------------------ |
| `/start`    | شروع و خوش‌آمدگویی |
| `/tools`    | لیست همه ابزارها   |
| `/salary`   | محاسبه حقوق        |
| `/tax`      | محاسبه مالیات      |
| `/loan`     | محاسبه وام         |
| `/currency` | مبدل ارز           |
| `/pdf`      | ابزارهای PDF       |
| `/image`    | ابزارهای تصویر     |
| `/text`     | ابزارهای متنی      |
| `/date`     | ابزارهای تاریخ     |
| `/calendar` | تقویم شمسی         |
| `/qrcode`   | تولید QR Code      |
| `/hash`     | تولید هش           |
| `/json`     | فرمت JSON          |
| `/base64`   | تبدیل Base64       |
| `/bgremove` | حذف پس‌زمینه       |
| `/password` | تولید رمز عبور     |
| `/rates`    | نرخ ارز            |

## امکانات

- دسترسی سریع به ۵۰+ ابزار
- دکمه‌های شیشه‌ای (Inline Keyboard)
- لینک مستقیم به ابزارها
- دریافت نرخ ارز
- دسته‌بندی ابزارها
