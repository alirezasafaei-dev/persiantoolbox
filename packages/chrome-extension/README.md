# جعبه ابزار فارسی - Chrome Extension

افزونه کروم برای دسترسی سریع به ابزارهای آنلاین فارسی.

## نصب

1. فایل‌های افزونه را دانلود کنید
2. در Chrome به `chrome://extensions/` بروید
3. Developer mode را فعال کنید
4. روی "Load unpacked" کلیک کنید
5. پوشه `packages/chrome-extension` را انتخاب کنید

## امکانات

- جستجوی سریع ابزارها
- دسترسی به ابزارهای محبوب
- نمایش ابزارهای اخیراً استفاده شده
- ۵۰+ ابزار در ۶ دسته‌بندی

## دسته‌بندی ابزارها

- PDF: تبدیل، فشرده‌سازی، ادغام، تقسیم، امنیت
- تصویر: حذف پس‌زمینه، تغییر اندازه، چرخش
- مالی: حقوق، مالیات، وام، سود، بیمه
- متنی: JSON، Base64، هش، شمارش کلمات
- تاریخ: تقویم شمسی، تبدیل تاریخ
- اعتبارسنجی: QR Code، رمز عبور
