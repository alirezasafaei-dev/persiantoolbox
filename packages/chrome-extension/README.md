# جعبه ابزار فارسی - Chrome Extension

افزونه کروم برای دسترسی سریع به ابزارهای آنلاین فارسی.

## نصب

1. فایل `persiantoolbox-extension.zip` را از حالت فشرده خارج کنید
2. در Chrome به `chrome://extensions/` بروید
3. Developer mode را فعال کنید
4. روی "Load unpacked" کلیک کنید
5. پوشه حاوی فایل‌های افزونه را انتخاب کنید

یا برای انتشار در Chrome Web Store:

1. `node build.js` را اجرا کنید
2. فایل `persiantoolbox-extension.zip` را در Chrome Web Store آپلود کنید

## امکانات

- جستجوی سریع ابزارها
- دسترسی به ابزارهای محبوب
- نمایش ابزارهای اخیراً استفاده شده
- ۸۰+ ابزار در ۱۰ دسته‌بندی

## دسته‌بندی ابزارها

- PDF: تبدیل، فشرده‌سازی، ادغام، تقسیم، امنیت (۱۷ ابزار)
- تصویر: حذف پس‌زمینه، تغییر اندازه، چرخش، OCR (۶ ابزار)
- مالی: حقوق، مالیات، وام، سود، بیمه، تورم (۲۶ ابزار)
- متنی: JSON، Base64، هش، شمارش کلمات (۱۱ ابزار)
- تاریخ: تقویم شمسی، تبدیل تاریخ، سن (۷ ابزار)
- اعتبارسنجی: QR Code، کد ملی، کارت بانکی (۸ ابزار)
- قرارداد: اجاره مسکونی، پیمانکاری ساختمان (۲ ابزار)
- کسب‌وکار: فاکتورساز و رسیدساز
- شغلی: رزومه‌ساز حرفه‌ای
- نگارش: ویرایشگر فارسی

## ساخت

```bash
node build.js
```
