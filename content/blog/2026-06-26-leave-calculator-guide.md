## چرا محاسبه حق مرخصی مهم است؟

بسیاری از کارمندان در ایران از حق قانونی مرخصی خود بی‌اطلاع هستند اا نمی‌دانند غیبت بدون مجوز چه تأثیری روی حقوق‌شان دارد. قانون کار ایران مرخصی‌های متنوعی را برای کارگران و کارمندان در نظر گرفته و محاسبه صحیح آنها می‌تواند تفاوت قابل توجهی در درآمد ماهانه ایجاد کند.

اگر ندانید چند روز مرخصی استحقاقی دارید، غیبت غیرموجه چقدر از حقوق‌تان کم می‌کند، یا مرخصی استعلاجی چه شرایطی دارد، ممکن است بخش قابل توجهی از درآمد خود را از دست بدهید. این راهنما کمک می‌کند حق مرخصی‌تان را به درستی محاسبه کنید.

## انواع مرخصی در قانون کار ایران

قانون کار ایران چند نوع مرخصی اصلی را تعریف کرده است:

| نوع مرخصی       | توضیح                                   | مدت قانونی                      |
| --------------- | --------------------------------------- | ------------------------------- |
| مرخصی استحقاقی  | مرخصی سالانه با حقوق                    | ۲۶ روز در سال (۱ روز در هفته)   |
| مرخصی استعلاجی  | مرخصی بیماری با تأیید پزشک              | تا ۳ روز در ماه (با تأیید پزشک) |
| مرخصی بدون حقوق | مرخصی با درخواست کارگر و موافقت کارفرما | توافقی                          |
| مرخصی ازدواج    | مرخصی به مناسبت ازدواج                  | ۳ روز                           |
| مرخصی فوت       | مرخصی به مناسبت فوت خانواده             | ۳ تا ۵ روز                      |
| مرخصی زایمان    | مرخصی برای مادران                       | ۹ ماه (از ۷ ماه با پزشک)        |

## حق مرخصی استحقاقی

مرخصی استحقاقی مهم‌ترین نوع مرخصی برای کارمندان است. بر اساس قانون کار:

- **۲۶ روز** مرخصی در سال (شامل روزهای جمعه)
- به ازای هر **۱ سال** سابقه کار، **۱ روز** مرخصی اضافی اضافه می‌شود (تا سقف معین)
- مرخصی استفاده نشده تا **۳ سال** قابل ذخیره است
- مرخصی ذخیره شده در پایان سال به صورت **حقوق نقدی** قابل پرداخت است

**نحوه محاسبه ارزش روزانه مرخصی:**

```
ارزش روزانه مرخصی = (حقوق پایه + مزایای مشمول) ÷ ۳۰
```

## تأثیر غیبت بر حقوق

غیبت بدون اجازه تأثیر مستقیمی روی حقوق دارد:

- برای هر روز غیبت، معادل **۱ روز حقوق** از فیش حقوقی کم می‌شود
- غیبت بیش از **۳ روز متوالی** بدون مجوز موجه می‌تواند منجر به **اخراج** شود
- غیبت غیرموجه بر ** سنوات و عیدی** نیز تأثیر منفی دارد
- غیبت مکرر در **سابقه کار** ثبت می‌شود و می‌تواند مانع دریافت بیمه بیکاری شود

## فرمول محاسبه حق مرخصی

### ۱. محاسبه ارزش روزانه مرخصی

```
ارزش روزانه مرfixی = حقوق پایه ÷ ۳۰
```

### ۲. محاسبه مرخصی استحقاقی سالانه

```
مرخصی استحقاقی = ۲۶ روز + (سابقه کار به سال)
```

### ۳. محاسبه کسر غیبت

```
کسر غیبت = ارزش روزانه مرخصی × تعداد روزهای غیبت
```

### ۴. محاسبه حق مرخصی ذخیره شده

```
حق مرخصی ذخیره = ارزش روزانه مرخصی × روزهای استفاده نشده
```

## مثال عملی: محاسبه حق مرخصی

فرض کنید یک کارمند با حقوق پایه **۲۵,۰۰۰,۰۰۰ تومان** و **۳ سال** سابقه کار:

### مراحل محاسبه

**گام ۱: محاسبه ارزش روزانه مرخصی**

```
ارزش روزانه مرfixی = ۲۵,۰۰۰,۰۰۰ ÷ ۳۰ = ۸۳۳,۳۳۳ تومان
```

**گام ۲: محاسبه مرخصی استحقاقی سالانه**

```
مرخصی استحقاقی = ۲۶ + ۳ = ۲۹ روز
```

**گام ۳: محاسبه ارزش کل مرخصی**

```
ارزش کل مرخصی = ۸۳۳,۳۳۳ × ۲۹ = ۲۴,۱۶۶,۶۶۷ تومان
```

**گام ۴: مثال غیبت**

اگر کارمند ۵ روز غیبت بدون مجوز داشته باشد:

```
کسر غیبت = ۸۳۳,۳۳۳ × ۵ = ۴,۱۶۶,۶۶۷ تومان
```

### خلاصه نتیجه

| شاخص                  | مبلغ / تعداد  |
| --------------------- | ------------- |
| حقوق پایه             | ۲۵,۰۰۰,۰۰۰    |
| مرخصی استحقاقی        | ۲۹ روز        |
| ارزش روزانه مرخصی     | ۸۳۳,۳۳۳       |
| ارزش کل مرخصی         | ۲۴,۱۶۶,۶۶۷    |
| روزهای غیبت           | ۵ روز         |
| کسر غیبت              | ۴,۱۶۶,۶۶۷     |
| **کسر نهایی از حقوق** | **۴,۱۶۶,۶۶۷** |

## استفاده از ابزار آنلاین ما

ابزار [محاسبه مرخصی PersianToolbox](/tools/leave-calculator) به شما کمک می‌کند:

1. **نوع مرخصی** را انتخاب کنید (استحقاقی، استعلاجی، بدون حقوق)
2. **حقوق پایه** و **مزایا** را وارد کنید
3. **تعداد روزهای مرخصی** یا **غیبت** را مشخص کنید
4. **سابقه کار** را وارد کنید
5. **نتیجه** را ببینید و با فیش حقوقی خود مقایسه کنید

**مزایای ابزار ما:**

- محاسبه دقیق بر اساس قوانین به‌روز ۱۴۰۴
- رایگان و بدون نیاز به ثبت‌نام
- تمام محاسبات در مرورگر شما انجام می‌شود (حریم خصوصی حفظ می‌شود)
- پشتیبانی از انواع مرخصی‌ها

## نکات مهم قانونی

### مرخصی استعلاجی

- فقط با تأیید پزشک قابل استفاده است
- حداکثر **۳ روز در ماه** با پرداخت حقوق (تجمعی تا سقف مشخص)
- فراتر از آن، مرخصی بدون حقوق محسوب می‌شود

### مرخصی ازدواج و فوت

- **ازدواج:** ۳ روز با حقوق
- **فوت خانواده درجه یک:** ۳ تا ۵ روز با حقوق
- **فوت والدین، همسر یا فرزند:** ۵ روز با حقوق

### ذخیره مرخصی

- مرخصی استفاده نشده تا **۳ سال** قابل ذخیره است
- در صورت ترک کار، مرخصی ذخیره شده به صورت **نقدی** پرداخت می‌شود
- مرخصی بیش از ۳ سال از بین می‌رود

## سوالات متداول

### آیا مرخصی استحقاقی شامل جمعه‌ها می‌شود؟

بله، مرخصی استحقاقی ۲۶ روز شامل تمام روزهای هفته از جمله جمعه‌ها می‌شود.

### غیبت چند روز منجر به اخراج می‌شود؟

غیبت بیش از **۳ روز متوالی** بدون مجوز موجه می‌تواند منجر به اخراج شود. اما کارفرما باید مراحل قانونی را رعایت کند.

### آیا مرخصی استعلاجی حقوق دارد؟

بله، حداکثر ۳ روز در ماه با تأیید پزشک. اگر بیش از این باشد، مرخصی بدون حقوق محسوب می‌شود.

### آیا می‌توانم مرخصی ذخیره شده را نقداً دریافت کنم؟

بله، در صورت ترک کار یا پایان قرارداد، مرخصی ذخیره شده تا ۳ سال به صورت نقدی پرداخت می‌شود.

### غیبت بر سنوات و عیدی تأثیر دارد؟

بله، غیبت غیرموجه بر محاسبه سنوات و عیدی نیز تأثیر منفی دارد.

### آیا ابزار محاسبه مرخصی رایگان است؟

بله، کاملاً رایگان و بدون نیاز به ثبت‌نام. تمام محاسبات در مرورگر شما انجام می‌شود.
