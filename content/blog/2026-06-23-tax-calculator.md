## محاسبه مالیات بر درآمد ۱۴۰۵

محاسبه‌گر مالیات PersianToolbox بر اساس قوانین سال ۱۴۰۵ طراحی شده است.

### نرخ مالیات ۱۴۰۵

| ردیف | درآمد مشمول مالیات        | نرخ |
| ---- | ------------------------- | --- |
| ۱    | تا ۴۰۰ میلیون تومان       | ۰٪  |
| ۲    | ۴۰۰ میلیون تا ۵۰۰ میلیون  | ۱۰٪ |
| ۳    | ۵۰۰ میلیون تا ۶۶۰ میلیون  | ۱۵٪ |
| ۴    | ۶۶۰ میلیون تا ۱ میلیارد   | ۲۰٪ |
| ۵    | ۱ میلیارد تا ۱.۳۳ میلیارد | ۲۵٪ |
| ۶    | بیش از ۱.۳۳ میلیارد       | ۳۰٪ |

### معافیت مالیاتی

- معافیت سالانه: ۴۰۰ میلیون تومان
- معافیت ماهانه: حدود ۳۳.۳ میلیون تومان

### نحوه محاسبه

1. حقوق ناخالص را وارد کنید
2. مزایا و بن‌ها را اضافه کنید
3. حق بیمه (۷٪) کسر شود
4. درآمد مشمول مالیات = حقوق ناخالص - بیمه
5. مالیات بر اساس جدول پلکانی محاسبه شود

### مثال

حقوق ناخالص: ۱۰۰ میلیون تومان

| ردیف | آیتم               | مبلغ       |
| ---- | ------------------ | ---------- |
| ۱    | حقوق ناخالص        | ۱۰۰ میلیون |
| ۲    | حق بیمه (۷٪)       | ۷ میلیون   |
| ۳    | درآمد مشمول مالیات | ۹۳ میلیون  |
| ۴    | معافیت مالیاتی     | ۴۰۰ میلیون |
| ۵    | مالیات قابل پرداخت | ۰ تومان    |

### نکات مهم

- محاسبات جنبه راهنمایی دارند
- برای محاسبات دقیق حتماً با حسابدار مشورت کنید
- قوانین مالیاتی ممکن است تغییر کنند
