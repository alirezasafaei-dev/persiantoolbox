## هزینه واقعی استخدام چقدر است؟

بسیاری از کارفرمایان فقط **حقوق پایه** کارمند را در نظر می‌گیرند، اما هزینه واقعی استخدام **۳۰ تا ۵۰ درصد بیشتر** از حقوق پایه است. شناخت این هزینه‌ها برای برنامه‌ریزی مالی کسب‌وکار ضروری است.

## اجزای هزینه واقعی استخدام

### ۱. بیمه کارفرما (۲۳٪)

کارفرمایان موظف به پرداخت **۲۳٪ حق بیمه** از حقوق پایه کارمند هستند:

| نوع بیمه      | نرخ     |
| ------------- | ------- |
| بازنشستگی     | ۱۴٪     |
| بیمه بیکاری   | ۳٪      |
| تأمین اجتماعی | ۳٪      |
| بیمه حوادث    | ۳٪      |
| **مجموع**     | **۲۳٪** |

### ۲. عیدانه و پاداش

طبق قانون کار، کارفرما موظف به پرداخت **حقوق ۱ ماه** به عنوان عیدانه در پایان هر سال است.

### ۳. سنوات (پایان خدمت)

اگر کارمند بیش از یک سال در شرکت کار کرده باشد، کارفرما موظف به پرداخت **حقوق هر سال یک ماه** به عنوان سنوات است.

### ۴. سایر مزایای اجباری

- **حق مسکن:** ۱۰٪ حداقل حقوق
- **حق خواربار:** ۱۰٪ حداقل حقوق
- **حق اولاد:** برای هر فرزند
- **بیمه تکمیلی:** در صورت وجود

## مثال عملی: کارمند با حقوق ۱۵ میلیون تومان

| هزینه                  | مبلغ ماهانه (تومان) |
| ---------------------- | ------------------- |
| حقوق پایه              | ۱۵,۰۰۰,۰۰۰          |
| بیمه کارفرما (۲۳٪)     | ۳,۴۵۰,۰۰۰           |
| عیدانه (تقسیم بر ۱۲)   | ۱,۲۵۰,۰۰۰           |
| سنوات (تقسیم بر ۱۲)    | ۱,۲۵۰,۰۰۰           |
| حق مسکن (۱۰٪)          | ۱,۵۰۰,۰۰۰           |
| **هزینه ماهانه واقعی** | **~۲۲,۴۵۰,۰۰۰**     |

**هزینه سالانه واقعی:** حدود **۲۷۰ میلیون تومان** به جای ۱۸۰ میلیون تومان حقوق پایه.

## مقایسه هزینه با کشورهای منطقه

| کشور    | هزینه بیمه کارفرما |
| ------- | ------------------ |
| ایران   | ۲۳٪                |
| ترکیه   | ۱۵.۵٪              |
| امارات  | ۱۲.۵٪              |
| عربستان | ۱۱.۷۵٪             |

## نکات مهم برای کارفرمایان

- **قرارداد کتبی** الزامی است؛ قرارداد شفاهی غیرقانونی است
- **حداقل حقوق** توسط شورای عالی کار تعیین می‌شود
- **بیمه** اجباری است و عدم پرداخت جریمه سنگین دارد
- **سنوات** در صورت ترک کار یا اخراج قابل پرداخت است
- هزینه استخدام **خارج از کشور** متفاوت است

## چرا از ابزار جعبه ابزار فارسی استفاده کنیم؟

ابزار محاسبه هزینه استخدام جعبه ابزار فارسی با در نظر گرفتن تمامی اجزای قانونی، هزینه واقعی استخدام هر کارمند را محاسبه می‌کند. این ابزار به شما کمک می‌کند بودجه دقیقی برای استخدام تنظیم کنید.
