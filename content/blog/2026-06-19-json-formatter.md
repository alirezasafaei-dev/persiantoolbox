## ابزار JSON Formatter

JSON (JavaScript Object Notation) فرمت استاندارد تبادل داده در وب است. ابزار JSON Formatter PersianToolbox به شما کمک می‌کند داده‌های JSON را فرمت، اعتبارسنجی و تبدیل کنید.

### امکانات ابزار

- **فرمت‌بندی خودکار**: JSON نامرتب را مرتب کنید
- **اعتبارسنجی**: خطاهای JSON را پیدا کنید
- **تبدیل به CSV**: JSON را به جدول تبدیل کنید
- **تبدیل به YAML**: JSON را به YAML تبدیل کنید
- **فشرده‌سازی**: حجم JSON را کاهش دهید

### کاربردها

- **API Testing**: بررسی پاسخ سرور
- **Database**: تبدیل داده پایگاه داده
- **Config Files**: فرمت فایل‌های پیکربندی
- **Data Migration**: انتقال داده بین سیستم‌ها

### نکات مهم

- JSON باید کلیدها را در雙 quotes بگذارد
- مقادیر string باید در dual quotes باشند
- اعداد نیاز به quotes ندارند
- true/false/null حروف خاص هستند
