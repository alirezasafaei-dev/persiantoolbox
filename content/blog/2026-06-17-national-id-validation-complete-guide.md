## کد ملی ایران چیست و چرا مهم است؟

هر شهروند ایرانی دارای یک **کد ملی ۱۰ رقمی** است که به عنوان شناسه یکتای هویتی در تمام تعاملات رسمی — از ثبت‌نام در سایت‌ها گرفته تا احراز هویت بانکی — استفاده می‌شود. این کد ساختار خاصی دارد و با یک الگوریتم ریاضی ساده اما مؤثر قابل بررسی است.

## الگوریتم بررسی صحت کد ملی

کد ملی ایران دقیقاً ۱۰ رقمی است. رقم آخر آن **رقم کنترلی** نامیده می‌شود که بر اساس ۹ رقم اول محاسبه می‌شود.

### فرمول محاسبه

```
وزن‌ها:  10  9  8  7  6  5  4  3  2
رقم‌ها:  d1  d2  d3  d4  d5  d6  d7  d8  d9  d10

جمع وزنی = (d1×10) + (d2×9) + (d3×8) + ... + (d9×2)
باقی‌مانده = جمع وزنی % 11

اگر باقی‌مانده < 2:
    رقم کنترل = باقی‌مانده
اگر باقی‌مانده >= 2:
    رقم کنترل = 11 - باقی‌مانده
```

### مثال عملی

فرض کنید کد ملی `0012345678` است:

```
وزن‌ها:  10  9  8  7  6  5  4  3  2
رقم‌ها:   0  0  1  2  3  4  5  6  7

جمع وزنی = (0×10) + (0×9) + (1×8) + (2×7) + (3×6) + (4×5) + (5×4) + (6×3) + (7×2)
         = 0 + 0 + 8 + 14 + 18 + 20 + 20 + 18 + 14
         = 112

باقی‌مانده = 112 % 11 = 2
رقم کنترل = 11 - 2 = 9

پس کد صحیح: 0012345678 (اگر رقم آخر 9 باشد)
```

## قوانین مهم اعتبارسنجی

علاوه بر الگوریتم رقم کنترلی، چند قانون دیگر هم باید رعایت شود:

1. **تعداد ارقام**: دقیقاً باید ۱۰ رقم باشد
2. **فقط عدد**: هیچ حرف یا کاراکتر خاصی نباید داشته باشد
3. **صفر اولیه**: برخی کدها با صفر شروع می‌شوند (معمولاً افراد متولد ۱۳۰۰ به بعد)
4. **اعداد تکراری**: کدهایی مثل `0000000000` نامعتبر هستند (تمام ارقام یکسان)

## چرا اعتبارسنجی کد ملی مهم است؟

### برای کسب‌وکارها

- **کاهش خطا**: جلوگیری از ثبت‌نام با کدهای اشتباه
- **احراز هویت**: تطبیق اطلاعات هویتی
- **合规**: رعایت الزامات قانونی سازمان‌ها

### برای توسعه‌دهندگان

- **API Integration**: استفاده از ابزارهای آنلاین در نرم‌افزارها
- **فرم‌های وب**: اعتبارسنجی سمت کلاینت قبل از ارسال
- **تست**: اطمینان از صحت داده‌های ورودی کاربر

## استفاده از ابزار آنلاین اعتبارسنجی

ابزار اعتبارسنجی کد ملی در جعبه ابزار فارسی کاملاً **آفلاین و محلی** عمل می‌کند:

- تمام پردازش‌ها در مرورگر شما انجام می‌شود
- هیچ داده‌ای به سرور ارسال نمی‌شود
- بدون نیاز به ثبت‌نام
- رایگان و بدون محدودیت

```javascript
// مثال ساده اعتبارسنجی کد ملی در JavaScript
function validateNationalId(code) {
  if (!/^\d{10}$/.test(code)) return false;
  if (/^(\d)\1{9}$/.test(code)) return false;

  const weights = [10, 9, 8, 7, 6, 5, 4, 3, 2];
  let sum = 0;
  for (let i = 0; i < 9; i++) {
    sum += parseInt(code[i]) * weights[i];
  }
  const remainder = sum % 11;
  const controlDigit = remainder < 2 ? remainder : 11 - remainder;

  return parseInt(code[9]) === controlDigit;
}
```

## سؤالات متداول

**آیا اعتبارسنجی سمت سرور لازم است؟**

بله. حتی اگر اعتبارسنجی سمت کلاینت انجام شود، همیشه باید در سمت سرور هم بررسی شود.

**آیا کد ملی با کد اقتصادی تفاوت دارد؟**

بله. کد اقتصادی مخصوص اشخاص حقوقی و شرکت‌ها است و ساختار متفاوتی دارد.

**آیا امکان تغییر کد ملی وجود دارد؟**

خیر. کد ملی یک شماره یکتا و غیرقابل تغییر است.
