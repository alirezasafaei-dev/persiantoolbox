## تقویم شمسی چیست؟

تقویم شمسی (هجری شمسی) تقویم رسمی ایران و افغانستان است. این تقویم بر اساس حرکت زمین به دور خورشید (سال نجومی) ساخته شده و دقیق‌ترین تقویم شناخته‌شده بشری است.

### ویژگی‌های تقویم شمسی

- **۱۲ ماه**: ۶ ماه ۳۱ روزه، ۵ ماه ۳۰ روزه، ۱ ماه ۲۹ (یا ۳۰) روزه
- **سال کبیسه**: هر ۴ سال یک روز اضافه (مانند میلادی)
- **نوروز**: شروع سال ۱ فروردین
- **ماه‌ها**: فروردین، اردیبهشت، خرداد، تیر، مرداد، شهریور، مهر، آبان، آذر، دی، بهمن، اسفند

### مقایسه با تقویم میلادی

| ویژگی           | شمسی      | میلادی        |
| --------------- | --------- | ------------- |
| روز شروع سال    | ۱ فروردین | ۱ ژانویه      |
| تعداد ماه‌ها    | ۱۲        | ۱۲            |
| روزهای سال      | ۳۶۵/۳۶۶   | ۳۶۵/۳۶۶       |
| سال شروع        | هجری شمسی | میلادی        |
| دقیق‌ترین تقویم | ✅        | ❌ (Gap year) |

## الگوریتم تبدیل تاریخ

تبدیل تاریخ شمسی به میلادی یکی از الگوریتم‌های پرکاربرد در برنامه‌نویسی فارسی است.

### مفاهیم کلیدی

1. **روز ژولین (Julian Day)**: یک سیستم شمارش روزهای پیوسته که پل ارتباطی بین دو تقویم است
2. **سال کبیسه شمسی**: قاعده خاصی برای تعیین سال‌های کبیسه دارد
3. **اختلاف زمانی**: تقریباً ۲۱ روز اختلاف بین شروع سال شمسی و میلادی

### مثال تبدیل

```python
def shamsi_to_gregorian(year, month, day):
    """تبدیل تاریخ شمسی به میلادی"""
    # الگوریتم ساده‌شده
    # سال میلادی = سال شمسی + 621 (تقریبی)
    # ماه و روز نیاز به محاسبه دقیق دارند

    gregorian_year = year + 621

    # محاسبه دقیق‌تر با در نظر گرفتن کبیسه
    # ...

    return f"{gregorian_year}/{month}/{day}"
```

## چرا تبدیل تاریخ مهم است؟

### برای برنامه‌نویسان

- **API‌های خارجی**: بسیاری از سرویس‌ها فقط تاریخ میلادی می‌پذیرند
- **پایگاه داده**: ذخیره‌سازی استاندارد تاریخ‌ها
- **گزارش‌ها**: تولید گزارش‌های دوزبانه

### برای کاربران عادی

- **فرم‌های اداری**: پر کردن فرم‌هایی که تاریخ میلادی می‌خواهند
- **سفر**: محاسبه تاریخ پروازها و رزرو هتل‌ها
- **مدارک بین‌المللی**: تطبیق تاریخ تولد در پاسپورت و ویزا

## نکات فنی مهم

### ۱. ناحیه زمانی (Timezone)

تاریخ‌ها باید با ناحیه زمانی صحیح ذخیره شوند. برای ایران: `Asia/Tehran` (UTC+3:30)

### ۲. فرمت‌های استاندارد

```javascript
// فرمت‌های رایج
const formats = {
  iso: '2026-06-26', // ISO 8601
  persian: '1405/04/05', // فرمت رایج شمسی
  persianFull: '۱۴۰۵/۰۴/۰۵', // فرمت فارسی با اعداد فارسی
  gregorian: 'June 26, 2026', // فرمت انگلیسی
};
```

### ۳. اعتبارسنجی تاریخ

همیشه قبل از ذخیره‌سازی، تاریخ را اعتبارسنجی کنید:

- آیا ماه معتبر است؟ (۱ تا ۱۲)
- آیا روز معتبر است؟ (بسته به ماه و سال کبیسه)
- آیا سال منطقی است؟

## سؤالات متداول

**آیا تقویم شمسی دقیق‌تر از میلادی است؟**

بله. تقویم شمسی بر اساس سال نجومی (حرکت زمین به دور خورشید) است، در حالی که تقویم میلادی از "سال قلابی" (Leap Year) استفاده می‌کند.

**اختلاف دقیق شمسی و میلادی چقدر است؟**

تقریباً ۲۱ روز. ۱ فروردین ۱۴۰۵ شمسی برابر با ۲۱ مارس ۲۰۲۶ میلادی است.

**آیا ابزارهای آنلاین تاریخ امن هستند؟**

ابزارهای PersianToolbox تمام محاسبات را در مرورگر شما انجام می‌دهند و هیچ داده‌ای به سرور ارسال نمی‌شود.
