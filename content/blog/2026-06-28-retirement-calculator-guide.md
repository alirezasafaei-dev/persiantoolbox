## چرا محاسبه بازنشستگی مهم است؟

بازنشستگی یکی از مهم‌ترین مراحل زندگی مالی هر فرد است. بدون برنامه‌ریزی دقیق، ممکن است در دوران بازنشستگی با مشکلات مالی جدی مواجه شوید. محاسبه سنوات خدمت، حق بیمه و مستمری بازنشستگی به شما کمک می‌کند تا بدانید دقیقاً چه مبلغی دریافت خواهید کرد.

در ایران، قوانین بازنشستگی پیچیدگی‌های خاص خود را دارد. درصد حق بیمه، سهم کارفرما، سنوات خدمت و نحوه محاسبه مستمری همگی عواملی هستند که باید به درستی درک شوند.

## اجزای اصلی بازنشستگی در ایران

### حق بیمه تأمین اجتماعی

در سیستم تأمین اجتماعی ایران، حق بیمه از دو بخش تشکیل شده است:

| بخش                   | درصد    | توضیح                      |
| --------------------- | ------- | -------------------------- |
| سهم بیمه‌شده (کارمند) | ۷٪      | از حقوق ناخالص کسر می‌شود  |
| سهم کارفرما           | ۲۳٪     | توسط کارفرما پرداخت می‌شود |
| **مجموع**             | **۳۰٪** | **کل حق بیمه**             |

**مثال عددی:**

- اگر حقوق ناخالص شما ۳۰,۰۰۰,۰۰۰ تومان باشد:
  - سهم شما: ۳۰,۰۰۰,۰۰۰ × ۷٪ = ۲,۱۰۰,۰۰۰ تومان
  - سهم کارفرما: ۳۰,۰۰۰,۰۰۰ × ۲۳٪ = ۶,۹۰۰,۰۰۰ تومان
  - مجموع حق بیمه: ۹,۰۰۰,۰۰۰ تومان

### سنوات خدمت

سنوات خدمت تعداد سال‌هایی است که فرد در یک یا چند محل کار مشمول بیمه تأمین اجتماعی بوده است. هر ماهی که حق بیمه پرداخت شود، یک ماه به سنوات اضافه می‌شود.

**شرایط بازنشستگی:**

| شرایط                 | توضیح                         |
| --------------------- | ----------------------------- |
| حداقل سنوات خدمت      | ۱۰ سال (۱۲۰ ماه)              |
| حداکثر سن بازنشستگی   | ۶۰ سال (مرد) - ۵۵ سال (زن)    |
| بازنشستگی پیش از موعد | ۲۰ سال خدمت و حداقل ۵۰ سال سن |

### مستمری بازنشستگی

مبلغ مستمری بازنشستگی بر اساس میانگین حقوق سه سال آخر خدمت محاسبه می‌شود. فرمول ساده:

```
مستمری = (میانگین حقوق ۳ سال آخر × ضریب مستمری) × درصد حق بیمه
```

## فرمول قانونی محاسبه مستمری

### گام ۱: محاسبه میانگین حقوق

```
میانگین حقوق = (حقوق سال اول + حقوق سال دوم + حقوق سال سوم) ÷ ۳
```

### گام ۲: اعمال ضریب مستمری

ضریب مستمری بر اساس سنوات خدمت تعیین می‌شود:

| سنوات خدمت    | ضریب مستمری |
| ------------- | ----------- |
| ۱۰ تا ۱۵ سال  | ۱.۰         |
| ۱۵ تا ۲۰ سال  | ۱.۱         |
| ۲۰ تا ۲۵ سال  | ۱.۲         |
| ۲۵ تا ۳۰ سال  | ۱.۳         |
| بیش از ۳۰ سال | ۱.۴         |

### گام ۳: اعمال درصد حق بیمه

درصد حق بیمه پرداختی تعیین می‌کند چه نسبتی از مستمری محاسبه شده به شما تعلق می‌گیرد:

- ۱۰ سال خدمت: ۳۵٪ مستمری
- ۲۰ سال خدمت: ۵۵٪ مستمری
- ۳۰ سال خدمت: ۷۵٪ مستمری
- ۳۵ سال خدمت: ۱۰۰٪ مستمری

## مثال عددی: کارمند ۲۵ سال خدمت

فرض کنید یک کارمند با ۲۵ سال سابقه خدمت قصد بازنشستگی دارد:

### اطلاعات ورودی

| پارامتر          | مبلغ             |
| ---------------- | ---------------- |
| حقوق سال آخر     | ۴۵,۰۰۰,۰۰۰ تومان |
| حقوق سال دوم آخر | ۴۰,۰۰۰,۰۰۰ تومان |
| حقوق سال سوم آخر | ۳۵,۰۰۰,۰۰۰ تومان |
| سنوات خدمت       | ۲۵ سال           |

### مراحل محاسبه

**گام ۱: محاسبه میانگین حقوق**

```
میانگین حقوق = (۴۵,۰۰۰,۰۰۰ + ۴۰,۰۰۰,۰۰۰ + ۳۵,۰۰۰,۰۰۰) ÷ ۳
میانگین حقوق = ۱۲۰,۰۰۰,۰۰۰ ÷ ۳ = ۴۰,۰۰۰,۰۰۰ تومان
```

**گام ۲: اعمال ضریب مستمری**

چون سنوات خدمت ۲۵ سال است، ضریب مستمری = ۱.۳

```
مستمری پایه = ۴۰,۰۰۰,۰۰۰ × ۱.۳ = ۵۲,۰۰۰,۰۰۰ تومان
```

**گام ۳: اعمال درصد حق بیمه**

چون ۲۵ سال خدمت دارد، درصد مستمری = ۶۵٪ (بین ۵۵٪ و ۷۵٪)

```
مستمری نهایی = ۵۲,۰۰۰,۰۰۰ × ۶۵٪ = ۳۳,۸۰۰,۰۰۰ تومان
```

### خلاصه نتیجه

| شاخص                    | مبلغ                 |
| ----------------------- | -------------------- |
| میانگین حقوق ۳ سال آخر  | ۴۰,۰۰۰,۰۰۰ تومان     |
| ضریب مستمری             | ۱.۳                  |
| مستمری پایه             | ۵۲,۰۰۰,۰۰۰ تومان     |
| درصد حق بیمه            | ۶۵٪                  |
| **مستمری نهایی ماهانه** | **۳۳,۸۰۰,۰۰۰ تومان** |

### مراحل استفاده

1. **اطلاعات پایه:** تعداد سال‌های خدمت و حقوق ماهانه را وارد کنید
2. **اطلاعات تکمیلی:** درصد حق بیمه پرداختی و ضرایب مرتبط را مشخص کنید
3. **محاسبه:** دکمه «محاسبه» را بزنید
4. **نتیجه:** مستمری تقریبی ماهانه و سالانه خود را مشاهده کنید

### مزایای ابزار ما

- **رایگان:** بدون نیاز به ثبت‌نام یا پرداخت
- **دقیق:** محاسبه بر اساس آخرین قوانین تأمین اجتماعی
- **سریع:** نتیجه در کمتر از ۱ ثانیه
- **امن:** تمام محاسبات در مرورگر شما انجام می‌شود

## نکات مهم قانونی

### افزایش سالانه مستمری

مستمری بازنشستگان هر ساله بر اساس مصوبه هیئت وزیران افزایش می‌یابد. معمولاً این افزایش بین ۱۵٪ تا ۲۵٪ است.

### حق بیمه اضافی

بیمه‌شدگان می‌توانند با پرداخت حق بیمه اضافی، سنوات خدمت خود را افزایش دهند. این کار باعث افزایش مستمری بازنشستگی می‌شود.

### بازنشستگی پیش از موعد

در صورت داشتن ۲۰ سال سابقه خدمت و حداقل ۵۰ سال سن، امکان بازنشستگی پیش از موعد وجود دارد. البته مستمری کمتری دریافت خواهد شد.
