## چرا محاسبه حقوق مهم است؟

بیشتر کارمندان در ایران وقتی فیش حقوقی‌شان را می‌بینند، با یک عدد غافلگیرکننده مواجه می‌شوند: مبلغی که به حساب‌شان واریز شده بسیار کمتر از آن چیزی است که انتظار داشتند. دلیلش ساده است: بیمه، مالیات، کسورات و مزایای مختلف، محاسبه را پیچیده می‌کنند.

اگر ندانید دقیقاً چه بخشی از حقوق‌تان به بیمه تأمین اجتماعی می‌رود، چه مبلغی مالیات دارد و اضافه کاری چطور محاسبه می‌شود، نمی‌توانید برنامه‌ریزی مالی دقیقی داشته باشید. این راهنما کمک می‌کند حقوق واقعی‌تان را بشناسید و بدانید هر ریال کجا می‌رود.

## اجزای حقوق در ایران

حقوق ماهانه یک کارمند ایرانی از چند بخش اصلی تشکیل شده است:

| جزء                | توضیح                                                        | مثال (تومان/ماه) |
| ------------------ | ------------------------------------------------------------ | ---------------- |
| حقوق پایه          | مبلغ پایه‌ای که بر اساس حداقل دستمزد یا قرارداد تعیین می‌شود | ۱۵,۰۶۶,۹۰۴       |
| کمک هزینه مسکن     | مبلغ مصوب سالانه                                             | ۳,۰۰۰,۰۰۰        |
| کمک هزینه غذا      | مبلغ مصوب سالانه                                             | ۲,۲۰۰,۰۰۰        |
| حق اولاد           | به ازای هر فرزند                                             | ۱,۶۶۲,۵۵۵        |
| حق تاهل            | برای کارمندان متأهل                                          | ۵۰۰,۰۰۰          |
| پایه سنوات         | سابقه کار ≥ ۱ سال                                            | ۵۰۰,۰۰۰          |
| کمک هزینه حمل‌ونقل | در صورت فعال بودن                                            | ۱,۵۰۰,۰۰۰        |
| بن کارگری          | در صورت فعال بودن                                            | ۲,۲۰۰,۰۰۰        |
| اضافه کاری         | ساعتی × ضریب                                                 | متغیر            |
| اضافه کاری شب      | ساعتی × ۱.۳۵                                                 | متغیر            |
| اضافه کاری تعطیل   | ساعتی × ۲.۰                                                  | متغیر            |
| حق مأموریت         | ۱۰٪ از روزانه                                                | متغیر            |

**کسورات اصلی:**

- **بیمه تأمین اجتماعی:** ۷٪ از حقوق ناخالص
- **مالیات حقوق:** پلکانی (از ۰٪ تا ۳۰٪) بر اساس درآمد مشمول مالیات

## نحوه محاسبه با ابزار ما

ابزار [محاسبه حقوق PersianToolbox](/salary) سه حالت محاسبه دارد:

### ۱. حقوق ناخالص به خالص

حقوق پایه و مزایا را وارد کنید تا حقوق خالص بعد از کسر بیمه و مالیات مشخص شود.

### ۲. حقوق خالص به ناخالص

مبلغ دریافتی مورد نظر را وارد کنید تا ببینید حقوق پایه چقدر باید باشد.

### ۳. حداقل دستمزد

بدون نیاز به ورودی، حداقل دستمزد قانونی با تمام مزایا و کسورات محاسبه می‌شود.

**مراحل استفاده:**

1. حالت محاسبه را انتخاب کنید (ناخالص به خالص یا برعکس)
2. حقوق پایه یا خالص مورد نظر را وارد کنید
3. سابقه کار، ساعات اضافه کاری و تعداد فرزند را مشخص کنید
4. در صورت نیاز، تنظیمات پیشرفته (تأهل، بن کارگری، حمل‌ونقل) را فعال کنید
5. دکمه «محاسبه مجدد» را بزنید

## مثال عملی: حقوق یک کارمند ۵ ساله

فرض کنید یک کارمند متأهل با ۲ فرزند و ۵ سال سابقه کار، حقوق پایه ۲۵,۰۰۰,۰۰۰ تومان دارد و ماهانه ۲۰ ساعت اضافه کاری عادی انجام می‌دهد.

### مراحل محاسبه

**گام ۱: محاسبه نرخ ساعتی**

```
نرخ ساعتی = حقوق پایه ÷ ۲۲۰
نرخ ساعتی = ۲۵,۰۰۰,۰۰۰ ÷ ۲۲۰ = ۱۱۳,۶۳۶ تومان
```

**گام ۲: محاسبه اضافه کاری**

```
اضافه کاری عادی = ۱۱۳,۶۳۶ × ۲۰ × ۱.۴ = ۳,۱۸۱,۸۱۸ تومان
```

**گام ۳: محاسبه مزایا**

| مبلغ (تومان)                  |
| ----------------------------- | -------------- |
| حقوق پایه: ۲۵,۰۰۰,۰۰۰         |
| کمک هزینه مسکن: ۳,۰۰۰,۰۰۰     |
| کمک هزینه غذا: ۲,۲۰۰,۰۰۰      |
| بن کارگری: ۲,۲۰۰,۰۰۰          |
| حق اولاد (×۲): ۳,۳۲۵,۱۱۰      |
| حق تاهل: ۵۰۰,۰۰۰              |
| پایه سنوات: ۵۰۰,۰۰۰           |
| کمک هزینه حمل‌ونقل: ۱,۵۰۰,۰۰۰ |
| اضافه کاری: ۳,۱۸۱,۸۱۸         |
| **مجموع ناخالص**              | **۴۱,۴۰۶,۹۲۸** |

**گام ۴: محاسبه کسورات**

```
بیمه = ۴۱,۴۰۶,۹۲۸ × ۷٪ = ۲,۸۹۸,۴۸۵ تومان
درآمد مشمول مالیات = ۴۱,۴۰۶,۹۲۸ - ۲,۸۹۸,۴۸۵ = ۳۸,۵۰۸,۴۴۳ تومان
```

چون درآمد مشمول مالیات (۳۸.۵ میلیون) کمتر از سقف معافیت (۴۰ میلیون) است، مالیات = **۰ تومان**.

```
مجموع کسورات = ۲,۸۹۸,۴۸۵ + ۰ = ۲,۸۹۸,۴۸۵ تومان
حقوق خالص = ۴۱,۴۰۶,۹۲۸ - ۲,۸۹۸,۴۸۵ = ۳۸,۵۰۸,۴۴۳ تومان
```

### خلاصه نتیجه

| شاخص               | مبلغ (تومان)   |
| ------------------ | -------------- |
| حقوق ناخالص        | ۴۱,۴۰۶,۹۲۸     |
| بیمه تأمین اجتماعی | ۲,۸۹۸,۴۸۵      |
| مالیات             | ۰              |
| مجموع کسورات       | ۲,۸۹۸,۴۸۵      |
| **حقوق خالص**      | **۳۸,۵۰۸,۴۴۳** |

## نکات مهم قانونی

### سقف معافیت مالیاتی

در سال ۱۴۰۴، درآمد مشمول مالیات تا **۴۰,۰۰۰,۰۰۰ تومان** در ماه معاف است. اگر بیمه را از ناخالص کم کنید و عدد زیر ۴۰ میلیون باشد، مالیاتی پرداخت نمی‌کنید.

### پلکان مالیاتی

| درآمد مشمول مالیات (تومان) | نرخ مالیات |
| -------------------------- | ---------- |
| تا ۴۰,۰۰۰,۰۰۰              | ۰٪         |
| ۴۰,۰۰۰,۰۰۱ تا ۵۰,۰۰۰,۰۰۰   | ۱۰٪        |
| ۵۰,۰۰۰,۰۰۱ تا ۶۶,۶۶۶,۶۶۷   | ۱۵٪        |
| ۶۶,۶۶۶,۶۶۸ تا ۱۰۰,۰۰۰,۰۰۰  | ۲۰٪        |
| ۱۰۰,۰۰۰,۰۰۱ تا ۱۳۳,۳۳۳,۳۳۳ | ۲۵٪        |
| بیش از ۱۳۳,۳۳۳,۳۳۳         | ۳۰٪        |

### ساعات کاری مبنای محاسبه

سالانه **۲,۶۴۰ ساعت** کاری مبنای محاسبه است. این یعنی ماهانه ۲۲۰ ساعت و روزانه تقریباً ۷.۳ ساعت.

### ضرایب اضافه کاری

| نوع اضافه کاری | ضریب |
| -------------- | ---- |
| عادی (روز)     | ۱.۴  |
| شب             | ۱.۳۵ |
| تعطیل          | ۲.۰  |

### سنوات و عیدی

- **پایه سنوات:** به ازای هر سال سابقه کار ≥ ۱ سال، مبلغ مصوب سالانه اضافه می‌شود
- **عیدی:** معادل ۲ ماه حقوق پایه (حداقل و حداکثر طبق مصوبه دولت)
- **پاداش:** بسته به سیاست شرکت متغیر است

### حق اولاد

به ازای هر فرزند تحت تکفل، مبلغ **۱,۶۶۲,۵۵۵ تومان** در ماه (سال ۱۴۰۴) به حقوق اضافه می‌شود.

## سوالات متداول

### آیا ابزار محاسبه حقوق رایگان است؟

بله، کاملاً رایگان و بدون نیاز به ثبت‌نام.

### آیا اطلاعات حقوق من ذخیره می‌شود؟

خیر. تمام محاسبات در مرورگر شما انجام می‌شود و هیچ داده‌ای به سرور ارسال نمی‌شود.

### آیا می‌توانم حقوق خالص را وارد کنم و حقوق ناخالص را ببینم؟

بله، حالت «حقوق خالص به ناخالص» دقیقاً همین کار را انجام می‌دهد. کافی است مبلغ دریافتی مورد نظر را وارد کنید.

### خروجی ابزار قطعی و حقوقی است؟

خروجی برای برآورد دقیق اولیه مناسب است. برای اعداد قطعی باید با قرارداد و قواعد رسمی محل کار تطبیق داده شود.

### آیا قوانین حقوقی به‌روز هستند؟

ابزار بر اساس قوانین مصوب سال ۱۴۰۴ کار می‌کند و در صورت تغییر مقررات باید نسخه قوانین به‌روزرسانی شود.

### اضافه کاری چطور محاسبه می‌شود؟

نرخ ساعتی = حقوق پایه ÷ ۲۲۰. سپس ضریب نوع اضافه کاری (عادی ۱.۴، شب ۱.۳۵، تعطیل ۲.۰) اعمال می‌شود.
