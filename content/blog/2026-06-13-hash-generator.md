## Hash چیست؟

Hash یک تابع ریاضی است که هر ورودی را به خروجی با طول ثابت تبدیل می‌کند. Hash غیرقابل برگشت است و برای امنیت استفاده می‌شود.

### الگوریتم‌های Hash

| الگوریتم | طول خروجی | امنیت | کاربرد   |
| -------- | --------- | ----- | -------- |
| MD5      | 128 bit   | پایین | checksum |
| SHA256   | 256 bit   | متوسط | رمزارز   |
| SHA512   | 512 bit   | بالا  | امنیت    |

### کاربردها

- **امنیت**: ذخیره رمز عبور
- ** checksum**: بررسی سلامت فایل
- ** رمزارز**: استخراج بیت‌کوین
- ** احراز هویت**: تأیید هویت

### نحوه استفاده

1. متن یا فایل را وارد کنید
2. الگوریتم Hash را انتخاب کنید
3. کد Hash تولید شده را کپی کنید
