## چرا امنیت رمز عبور مهم است؟

رمزهای عبور ضعیف بزرگ‌ترین تهدید امنیتی آنلاین هستند. ابزار تولید رمز عبور PersianToolbox کمک می‌کند تا رمزهای قوی و امن بسازید.

### ویژگی‌های رمز عبور قوی

- **طول مناسب**: حداقل ۱۲ کاراکتر
- **ترکیب حروف**: بزرگ، کوچک و اعداد
- **نمادها**: استفاده از کاراکترهای خاص
- **بدون اطلاعات شخصی**: بدون نام یا تاریخ تولد
- **یکتا برای هر حساب**: استفاده از رمز متفاوت

### نکات امنیتی مهم

- از رمز عبور یکسان برای چند حساب استفاده نکنید
- رمزهای عبور را به‌صورت متنی ذخیره نکنید
- از احراز هویت دو مرحله‌ای استفاده کنید
- رمزهای عبور را به‌صورت دوره‌ای تغییر دهید

### ابزار تولید رمز عبور

با استفاده از ابزار رمز عبور PersianToolbox می‌توانید رمزهای امن و تصادفی بسازید. این ابزار به‌صورت محلی در مرورگر کار می‌کند و هیچ اطلاعاتی ارسال نمی‌شود.
