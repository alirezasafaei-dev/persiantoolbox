# نحوه نوشتن فاکتور حرفه‌ای؛ ساختار، اقلام ضروری و اشتباهات رایج

فاکتور حرفه‌ای نه تنها یک سند مالی است، بلکه نمایانگر اعتبار و حرفه‌ای بودن کسب‌وکار شماست. در این راهنما، نحوه نوشتن فاکتور استاندارد را آموزش می‌دهیم.

## ساختار استاندارد فاکتور

### اقلام ضروری فاکتور

| ردیف | قلم                    | توضیح                          |
| ---- | ---------------------- | ------------------------------ |
| ۱    | **شماره فاکتور**       | شماره یکتای فاکتور برای پیگیری |
| ۲    | **تاریخ صدور**         | تاریخ صدور فاکتور              |
| ۳    | **اطلاعات فروشنده**    | نام، آدرس، شماره تماس          |
| ۴    | **اطلاعات خریدار**     | نام، آدرس، شماره تماس          |
| ۵    | **شرح خدمات/کالا**     | توضیح دقیق کالا یا خدمات       |
| ۶    | **تعداد و واحد**       | تعداد و واحد شمارش             |
| ۷    | **مبلغ واحد**          | قیمت هر واحد                   |
| ۸    | **جمع کل**             | مبلغ کل پیش از مالیات          |
| ۹    | **مالیات ارزش افزوده** | در صورت مشمول بودن             |
| ۱۰   | **جمع نهایی**          | مبلغ قابل پرداخت               |
| ۱۱   | **شرایط پرداخت**       | مهلت و نحوه پرداخت             |
| ۱۲   | **امضا و مهر**         | مهر و امضای فروشنده            |

## تفاوت فاکتور کالا و خدمات

### فاکتور کالا

- شامل **ارزش افزوده** (VAT) می‌شود
- شماره مالیاتی فروشنده الزامی است
- اطلاعات خریدار (شماره اقتصادی) در صورت نیاز

### فاکتور خدمات

- ممکن است از مالیات معاف باشد (بسته به نوع خدمات)
- شرح خدمات باید دقیق و واضح باشد
- ذکر **شرایط گارانتی** (در صورت وجود)

## اشتباهات رایج در صدور فاکتور

### ۱. عدم ذکر شماره فاکتور

**اشتباه**: صدور فاکتور بدون شماره یکتا.
**درست**: هر فاکتور باید شماره منحصربفرد داشته باشد.

### ۲. عدم ذکر شرایط پرداخت

**اشتباه**: ذکر نکردن مهلت و نحوه پرداخت.
**درست**: شرایط پرداخت باید در فاکتور ذکر شود.

### ۳. عدم درج اطلاعات کامل طرفین

**اشتباه**: ناقص بودن اطلاعات فروشنده یا خریدار.
**درست**: اطلاعات کامل شامل نام، آدرس و شماره تماس.

### ۴. عدم ذکر تاریخ سررسید

**اشتباه**: ذکر نکردن تاریخ سررسید پرداخت.
**درست**: تاریخ سررسید باید مشخص باشد.

## چک‌لیست صدور فاکتور

- [ ] شماره فاکتور یکتا درج شده
- [ ] تاریخ صدور مشخص است
- [ ] اطلاعات فروشنده کامل است
- [ ] اطلاعات خریدار کامل است
- [ ] شرح خدمات/کالا دقیق است
- [ ] تعداد و مبلغ واحد درج شده
- [ ] مالیات محاسبه شده (در صورت مشمول بودن)
- [ ] جمع نهایی مشخص است
- [ ] شرایط پرداخت ذکر شده
- [ ] امضا و مهر اضافه شده
