## چرا هزینه استخدام با حقوق پایه فرق دارد؟

وقتی یک کارفرما می‌خواهد کارمند جدیدی استخدام کند، اولین چیزی که به ذهنش می‌رسد حقوق پایه است. اما هزینه واقعی استخدام هرگز به همین عدد ختم نمی‌شود. بیمه کارفرما، عیدی، سنوات، حق مسکن و ده‌ها مورد دیگر، هزینه نهایی را ۳۰ تا ۵۰ درصد بالاتر از حقوق پایه می‌برد.

اگر بودجه استخدام را فقط بر اساس حقوق پایه تنظیم کنید، در پایان سال با کسری شدید مواجه خواهید شد. این راهنما کمک می‌کند هزینه واقعی استخدام را دقیق محاسبه کنید.

## تمام هزینه‌های جانبی استخدام

### جدول کامل اجزای هزینه

| هزینه        | توضیح                        | نرخ/مبلغ               |
| ------------ | ---------------------------- | ---------------------- |
| حقوق پایه    | مبلغ قرارداد                 | متغیر                  |
| بیمه کارفرما | سهم کارفرما به تأمین اجتماعی | ۲۳٪ از حقوق پایه       |
| عیدی         | حقوق پایان سال               | حداقل ۲ ماه حقوق       |
| سنوات        | حق پایان خدمت                | هر سال ۱ ماه حقوق      |
| حق مسکن      | مصوب سالانه                  | حداقل ۱۰٪ حداقل دستمزد |
| حق خواربار   | مصوب سالانه                  | حداقل ۱۰٪ حداقل دستمزد |
| حق اولاد     | به ازای هر فرزند             | مبلغ مصوب سالانه       |
| بیمه تکمیلی  | در صورت ارائه                | متغیر                  |

### ۲۳٪ سهم بیمه کارفرما

کارفرمایان موظف به پرداخت **۲۳٪ حق بیمه** از محل حقوق پایه کارمند به سازمان تأمین اجتماعی هستند. این مبلغ شامل بخش‌های زیر است:

| نوع بیمه      | نرخ     |
| ------------- | ------- |
| بازنشستگی     | ۱۴٪     |
| بیمه بیکاری   | ۳٪      |
| تأمین اجتماعی | ۳٪      |
| بیمه حوادث    | ۳٪      |
| **مجموع**     | **۲۳٪** |

**نکته مهم:** ۷٪ حق بیمه کارمند از حقوق خالص کارمند کم می‌شود، اما ۲۳٪ سهم کارفرما هزینه اضافی برای کارفرماست. یعنی هزینه واقعی استخدام از این ۲۳٪ شروع می‌شود.

### عیدی و پاداش

طبق ماده ۹۱ قانون کار، کارفرما موظف به پرداخت **حقوق ۱ ماه** به عنوان عیدی در پایان هر سال است. البته در عمل، عیدی معمولاً معادل **۲ ماه حقوق پایه** (با احتساب حداقل و حداکثر مصوبه دولت) پرداخت می‌شود.

**محاسبه ماهانه عیدی:**

```
عیدی ماهانه = (حقوق پایه × ۲) ÷ ۱۲
```

برای کارمند با حقوق ۱۵ میلیون:

```
عیدی ماهانه = (۱۵,۰۰۰,۰۰۰ × ۲) ÷ ۱۲ = ۲,۵۰۰,۰۰۰ تومان
```

### سنوات و حق اخراج

سنوات (حق پایان خدمت) به ازای **هر سال سابقه کار**، معادل **یک ماه حقوق پایه** به کارمند پرداخت می‌شود. این مبلغ در صورت ترک کار، اخراج یا بازنشستگی قابل پرداخت است.

**محاسبه ماهانه سنوات:**

```
سنوات ماهانه = حقوق پایه ÷ ۱۲
```

برای کارمند با حقوق ۱۵ میلیون:

```
سنوات ماهانه = ۱۵,۰۰۰,۰۰۰ ÷ ۱۲ = ۱,۲۵۰,۰۰۰ تومان
```

**نکته:** اگر کارمند ۳ سال سابقه کار داشته باشد، مبلغ سنوات معادل ۳ ماه حقوق خواهد بود.

## مثال عددی: استخدام کارمند با حقوق ۱۵ میلیون تومان

فرض کنید می‌خواهید یک کارمند با حقوق پایه **۱۵,۰۰۰,۰۰۰ تومان** استخدام کنید. بیایید هزینه واقعی ماهانه و سالانه را محاسبه کنیم.

### محاسبه هزینه ماهانه

| هزینه                  | مبلغ ماهانه (تومان) | نحوه محاسبه           |
| ---------------------- | ------------------- | --------------------- |
| حقوق پایه              | ۱۵,۰۰۰,۰۰۰          | قرارداد               |
| بیمه کارفرما (۲۳٪)     | ۳,۴۵۰,۰۰۰           | ۱۵,۰۰۰,۰۰۰ × ۲۳٪      |
| عیدی (تقسیم بر ۱۲)     | ۲,۵۰۰,۰۰۰           | (۱۵,۰۰۰,۰۰۰ × ۲) ÷ ۱۲ |
| سنوات (تقسیم بر ۱۲)    | ۱,۲۵۰,۰۰۰           | ۱۵,۰۰۰,۰۰۰ ÷ ۱۲       |
| حق مسکن                | ۱,۵۰۰,۰۰۰           | ۱۰٪ حداقل دستمزد      |
| حق خواربار             | ۱,۵۰۰,۰۰۰           | ۱۰٪ حداقل دستمزد      |
| **هزینه ماهانه واقعی** | **~۲۵,۲۰۰,۰۰۰**     | مجموع                 |

### محاسبه هزینه سالانه

| هزینه                  | مبلغ سالانه (تومان) |
| ---------------------- | ------------------- |
| حقوق پایه              | ۱۸۰,۰۰۰,۰۰۰         |
| بیمه کارفرما           | ۴۱,۴۰۰,۰۰۰          |
| عیدی                   | ۳۰,۰۰۰,۰۰۰          |
| سنوات                  | ۱۵,۰۰۰,۰۰۰          |
| حق مسکن                | ۱۸,۰۰۰,۰۰۰          |
| حق خواربار             | ۱۸,۰۰۰,۰۰۰          |
| **هزینه سالانه واقعی** | **~۳۰۲,۴۰۰,۰۰۰**    |

**نتیجه:** هزینه واقعی سالانه استخدام یک کارمند با حقوق ۱۵ میلیون، تقریباً **۳۰۲ میلیون تومان** است — یعنی حدود **۶۸ درصد بیشتر** از حقوق پایه.

## مقایسه هزینه با کشورهای منطقه

| کشور    | هزینه بیمه کارفرما | سایر هزینه‌های اجباری  |
| ------- | ------------------ | ---------------------- |
| ایران   | ۲۳٪                | عیدی + سنوات + حق مسکن |
| ترکیه   | ۱۵.۵٪              | بیمه بازنشستگی         |
| امارات  | ۱۲.۵٪              | EOSB (سنوات)           |
| عربستان | ۱۱.۷۵٪             | GOSI                   |

## نکات مهم برای کارفرمایان

1. **قرارداد کتبی الزامی است** — قرارداد شفاهی غیرقانونی است و در صورت بروز اختلاف، کارفرما محکوم می‌شود
2. **بیمه اجباری است** — عدم پرداخت حق بیمه جریمه سنگین و حتی تعطیلی کسب‌وکار به همراه دارد
3. **سنوات قابل پرداخت است** — حتی در صورت استعفای کارمند، سنوات باید پرداخت شود
4. **هزینه استخدام خارج از کشور متفاوت است** — قوانین بیمه و مالیات هر کشور متفاوت است
5. **بودجه‌بندی دقیق** — همیشه ۳۰ تا ۵۰ درصد بالاتر از حقوق پایه بودجه در نظر بگیرید

## استفاده از ابزار آنلاین

ابزار [محاسبه هزینه استخدام PersianToolbox](/tools/hiring-cost) با در نظر گرفتن تمامی اجزای قانونی، هزینه واقعی استخدام هر کارمند را محاسبه می‌کند.

**مراحل استفاد:**

1. حقوق پایه کارمند را وارد کنید
2. سابقه کار و تعداد فرزندان را مشخص کنید
3. گزینه‌های اضافی (بیمه تکمیلی، حق مسکن و...) را فعال کنید
4. دکمه «محاسبه» را بزنید
5. هزینه ماهانه و سالانه واقعی را ببینید

**مزایای ابزار:**

- محاسبه خودکار ۲۳٪ بیمه کارفرما
- احتساب عیدی و سنوات
- در نظر گرفتن حق مسکن و خواربار
- خروجی جزئیات هر بخش
- کاملاً رایگان و بدون ثبت‌نام

## سوالات متداول

### آیا بیمه کارفرما اجباری است؟

بله. طبق قانون تأمین اجتماعی، پرداخت ۲۳٪ حق بیمه توسط کارفرما اجباری است. عدم پرداخت مشمول جریمه و تعقیب قانونی می‌شود.

### آیا عیدی فقط در پایان سال پرداخت می‌شود؟

خیر. از نظر محاسباتی، عیدی را می‌توان به صورت ماهانه کنار گذاشت تا در پایان سال با مشکل بودجه مواجه نشوید.

### سنوات شامل چه کسانی می‌شود؟

کلیه کارمندانی که بیش از یک سال در یک محل کار مشغول به کار بوده باشند. حتی در صورت استعفا، سنوات قابل پرداخت است.

### آیا حق مسکن و خواربار اجباری است؟

بله. حق مسکن و حق خواربار حداقل ۱۰ درصد حداقل دستمزد مصوب شورای عالی کار است و پرداخت آن اجباری می‌باشد.

### هزینه استخدام یک کارمند نیمه‌وقت چقدر است؟

هزینه استخدام نیمه‌وقت نیز شامل ۲۳٪ بیمه کارفرما می‌شود، اما عیدی و سنوات بر اساس ساعات کاری محاسبه می‌شود.

### آیا می‌توان هزینه استخدام را کاهش داد؟

استفاده از قراردادهای پیمانکاری یا دورکاری می‌تواند بخشی از هزینه‌های جانبی را کاهش دهد، اما بیمه کارفرما همچنان اجباری است.
