## آموزش کار با JSON

ابزار JSON PersianToolbox امکان فرمت‌دهی و اعتبارسنجی کد JSON را فراهم می‌کند.

### JSON چیست؟

JSON (JavaScript Object Notation) یک فرمت سبک برای تبادل داده است.

### کاربردها

- انتقال داده بین سرور و کلاینت
- ذخیره‌سازی تنظیمات
- APIهای REST
- فایل‌های پیکربندی

### نحوه استفاده

1. کد JSON را وارد کنید
2. روی فرمت کلیک کنید
3. کد فرمت شده را کپی کنید

### ویژگی‌ها

- فرمت‌دهی خودکار
- اعتبارسنجی ساختار
- نمایش خطاها
- پشتیبانی از فارسی

### نکات مهم

- از کوتیشن دوگانه استفاده کنید
- کلیدها باید رشته باشند
- مقادیر نباید با کاما ختم شوند
