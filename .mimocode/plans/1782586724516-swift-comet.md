# SEO & Organic Traffic Growth Plan

## Goal

Maximize PersianToolbox's organic traffic from Google and Bing by improving structured data, internal linking, content quality, and technical SEO.

## Current State

- 372 URLs in sitemap
- 54 blog articles (dates spread)
- 5+ guides with Article schema
- 80+ tool pages with JSON-LD
- BlogPostSchema with Article type
- BreadcrumbSchema on all pages
- FaqSchema on tool pages
- Sitemap with priorities and changefreq

## Gap Analysis

### Critical Gaps

1. **No HowTo schema on guides** — guides have Article schema but not HowTo, missing rich snippet opportunity
2. **Weak internal linking** — blog posts don't link to each other, tools don't cross-link
3. **No dateModified on blog posts** — Google sees all posts as same age
4. **Blog posts are thin** — many posts are short, need pillar content

### Medium Gaps

5. **No category page optimization** — category pages lack structured data
6. **No "ابزارهای مرتبط" on category pages** — missing cross-linking
7. **RSS feed exists but not optimized** — no self-referencing link
8. **Blog posts lack author markup** — using Organization instead of Person

### Low Priority

9. **No video schema** — could add for future video content
10. **No review/rating schema** — could add for tool comparisons

## Implementation Plan

### Task 1: Add HowTo Schema to Guides

**File:** `app/guides/[slug]/page.tsx`

- Change Article schema to HowTo schema with steps
- Add `mainEntity` with HowToStep items from guide sections
- Keep existing FAQ and breadcrumb schemas

### Task 2: Improve Blog Internal Linking

**Files:** `app/blog/[slug]/page.tsx`

- Add "مقالات مرتبط" section that links to 3 related posts by tag matching
- Add "ابزارهای مرتبط" section that links to tools mentioned in the post
- Use existing BlogToolCTA component for tool links

### Task 3: Add dateModified to Blog Posts

**Files:** `lib/blog.ts`, `components/seo/BlogPostSchema.tsx`

- Add `modifiedDate` field to blog frontmatter
- Pass `modifiedDate` to BlogPostSchema
- Default to `date` if not specified

### Task 4: Create Pillar Content Pages

**Files:** Create 3 new pillar pages in `content/blog/`

- "راهنمای جامع ابزارهای مالی فارسی" — links to all financial tools
- "راهنمای جامع ابزارهای PDF فارسی" — links to all PDF tools
- "راهنمای جامع ابزارهای تاریخ فارسی" — links to all date tools
- Each pillar: 1500+ words, links to 10+ tools, has FAQ

### Task 5: Optimize Category Pages

**Files:** `app/(tools)/*/page.tsx`, `components/seo/ToolSeoContent.tsx`

- Add `ItemList` schema to category pages
- Add "ابزارهای مرتبط" section linking to other categories
- Improve category descriptions for long-tail keywords

### Task 6: Add RSS Self-Referencing Link

**Files:** `app/layout.tsx`

- Add `<link rel="alternate" type="application/rss+xml" ...>` to head
- Points to `/feed.xml`

### Task 7: Improve Blog Post Schema

**Files:** `components/seo/BlogPostSchema.tsx`

- Add `author` as Person schema (not Organization)
- Add `wordCount` to Article schema
- Add `articleSection` (category) to Article schema

### Task 8: Add Breadcrumb to Blog Posts

**Files:** `app/blog/[slug]/page.tsx`

- Add BreadcrumbSchema component (may already exist)
- Ensure breadcrumb: Home → Blog → Category → Post
