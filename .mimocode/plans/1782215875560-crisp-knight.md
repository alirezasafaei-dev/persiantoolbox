## PHASE 0: PRODUCT VISION

### Who We Serve

- **Primary**: Persian-speaking users in Iran who need quick, accurate financial calculations (salary, loans, tax)
- **Secondary**: Professionals who need PDF/image/text tools daily (resume, signature, OCR)
- **Tertiary**: Developers and businesses looking for reliable, privacy-first tools

### Jobs They Hire Us For

1. "What is my net salary after tax and insurance?" (salary calculator)
2. "How much is my monthly loan payment?" (loan calculator)
3. "Convert this PDF to images" (PDF tools)
4. "Compress this image for web" (image tools)
5. "Convert this Shamsi date to Gregorian" (date tools)
6. "Count words in my Persian text" (text tools)

### Why Trust Us vs Alternatives

- **100% local processing** — no data leaves the browser
- **No signup required** — instant access
- **Persian-first** — RTL, Persian numbers, native UI
- **Accurate financial logic** — based on real Iranian labor laws
- **Transparent** — open about how calculations work

### Growth Strategy

- Organic search via SEO-optimized tool pages
- Word of mouth from trust and quality
- Blog content for long-tail keywords
- Internal linking between tools and categories

## PHASE 1: MULTI-AGENT AUDIT FINDINGS

### 1. Product & Strategy Audit

**Current State**: 59 tools in 6 categories, 17 blog posts, 421 unit tests + 43 E2E tests.

**Strengths**:

- Comprehensive tool coverage across financial, PDF, image, text, date, and validation categories
- Strong privacy-first positioning (100% client-side processing)
- Good SEO foundation with structured data (JSON-LD, breadcrumbs, FAQ schema)

**Critical Gaps**:

- No clear value proposition on homepage for "why use us" beyond tool listing
- Missing trust signals (enamad seal exists but not prominently featured)
- No user onboarding or first-use guidance
- Tool discovery is weak — search is basic, no "recently used" or "recommended for you"
- No comparison tool for financial products (e.g., "which loan type is best?")
- Blog content is good but lacks internal linking to tools

**Priority Items**:

1. Homepage rebuild as trust & discovery engine
2. Tool page improvements with better CTAs

### 3. Architecture & Backend Audit

**Project Structure**:

**API Routes**:

- Auth system with sessions, CSRF, rate limiting
- PostgreSQL for persistent data

**Technical Debt**:

- ❌ Multiple deploy scripts (deploy-vps-auto.sh, deploy-vps.sh, quick-deploy.sh, copy-project-to-vps.sh) — should consolidate

**Security**:

- ✅ CSRF protection

- ✅ Rate limiting

**Interest Calculator** (`features/interest/`):

- ✅ Simple and compound interest

**Currency Converter** (`/tools/currency-converter`):

- ⚠️ Need to verify how exchange rates are sourced/updated

**Tax Calculator** (`/tools/tax-calculator`):

- ⚠️ Uses same salary-laws data — should be consistent

**Currency Handling**:

- All values in Toman (user-facing)
- Conversion: 1 Toman = 10 Rial (used internally where needed)
- Integer-safe arithmetic in loan logic
- ⚠️ Not all tools use consistent rounding — some use Math.round, some use Math.floor
