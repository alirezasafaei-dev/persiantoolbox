**تاریخ**: 2026-06-22
**وضعیت**: Plan
**اولویت**: داشبورد ادمین → بلاگ

---

## خلاصه اجرایی

### هدف

ایجاد یک داشبورد ادمین جامع و حرفه‌ای برای مدیریت سایت PersianToolbox، به همراه یک سیستم بلاگ ساده و کارآمد.

### انتخاب‌ها

- **داشبورد**: Analytics + Content Management + Ops + Site Settings
- **بلاگ**: مقالات آموزشی + اخبار + SEO
- **محتوا**: Markdown files در پوشه `content/blog/`
- **اولویت**: داشبورد ادمین اول، سپس بلاگ

## فاز ۱: داشبورد ادمین (اولویت اول)

- ایجاد shared admin shell با sidebar navigation
- محافظت از تمام مسیرهای `/admin/*` با احراز هویت
- Sidebar با لینک‌های: Dashboard, Analytics, Content, Settings, Ops

## نکات فنی مهم

### RTL Support

- تمام UI به فارسی و RTL باشد
- از CSS logical properties استفاده شود
- فونت Vazirmatn برای تمام متن‌ها

### Dark Mode

- تمام کامپوننت‌ها از CSS variables استفاده کنند
- پشتیبانی کامل از dark mode

### Privacy-First

- تمام پردازش‌ها محلی باشد
- داده‌های analytics فقط در سرور ذخیره شوند
- لاگ‌ها حاوی اطلاعات حساس نباشند

### Performance

- Lazy loading برای کامپوننت‌های سنگین
- Static generation برای صفحات بلاگ
- No external dependencies برای نمودارها
